# Zero 语言规范

## 概述

Zero是一种简洁、现代的编程语言，专注于易学性和实用性。

## 语法规范

### 1. 变量声明

```zero
// 不可变变量
let x = 42;
let name = "Alice";

// 可变变量
var count = 0;
count = count + 1;
```

### 2. 数据类型

#### 基本类型

- **整数**: `42`, `-100`
- **浮点数**: `3.14`, `-2.5`
- **字符串**: `"Hello, World!"`
- **布尔值**: `true`, `false`

### 3. 运算符

#### 算术运算符

- `+` 加法
- `-` 减法
- `*` 乘法
- `/` 除法
- `%` 取模

#### 比较运算符

- `==` 等于
- `!=` 不等于
- `<` 小于
- `<=` 小于等于
- `>` 大于
- `>=` 大于等于

#### 逻辑运算符

- `&&` 逻辑与
- `||` 逻辑或
- `!` 逻辑非

### 4. 控制流

#### If-Else 语句

```zero
if condition {
    // 代码块
} else {
    // 代码块
}
```

#### While 循环

```zero
while condition {
    // 代码块
}
```

#### For 循环

```zero
for i in 0..10 {
    // 代码块
}
```

### 5. 函数

```zero
fn function_name(param1, param2) {
    // 函数体
    return result;
}
```

#### 函数调用

```zero
let result = function_name(arg1, arg2);
```

### 6. 注释

```zero
// 单行注释
```

### 7. 输出

```zero
print("Hello, World!");
print(variable);
```

## 关键字列表

- `let` - 声明不可变变量
- `var` - 声明可变变量
- `fn` - 函数声明
- `return` - 返回值
- `if` - 条件语句
- `else` - 条件分支
- `while` - While循环
- `for` - For循环
- `in` - For循环中的范围运算符
- `true` - 布尔真值
- `false` - 布尔假值
- `print` - 输出函数

## 语法约定

1. 语句以分号 `;` 结束
2. 代码块使用大括号 `{}` 包围
3. 函数参数用逗号 `,` 分隔
4. 范围使用 `..` 表示（不包含结束值）

## 类型推断

Zero支持类型推断，编译器会根据赋值自动推断变量类型：

```zero
let x = 42;        // 推断为整数
let y = 3.14;      // 推断为浮点数
let s = "hello";   // 推断为字符串
let b = true;      // 推断为布尔值
```

## 作用域规则

- 变量在声明的代码块内有效
- 内层作用域可以访问外层作用域的变量
- 同名变量会产生遮蔽（shadowing）

```zero
let x = 10;
{
    let x = 20;  // 内层的x遮蔽外层的x
    print(x);    // 输出: 20
}
print(x);        // 输出: 10
```
